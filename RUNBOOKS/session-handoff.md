# Session Handoff Runbook

Use this runbook whenever an agent or human ends a working session.

## Steps

1. Summarize what changed, what is pending, and what is blocked.
2. Link updated artifacts in:
   - [PROJECTS](../PROJECTS/)
   - [POLICIES](../POLICIES/)
   - [DOCS/SHOW/episodes](../DOCS/SHOW/episodes/)
3. Record risk decisions and unresolved safety questions.
4. Confirm no secrets or personal information were introduced.
5. Assign the next owner and target date in ISO format (`YYYY-MM-DD`).
